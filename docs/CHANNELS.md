# Channels

## Telegram
- Set `TELEGRAM_BOT_TOKEN` in `.env`.
- Enable Telegram in `config/agent.json`.

## Discord / Slack
Stubs exist, but SDKs are not installed by default. Add `discord.js` or `@slack/bolt` and implement handlers.

## WebChat
Enabled by default. Open `http://localhost:18789/webchat` after starting the gateway.
